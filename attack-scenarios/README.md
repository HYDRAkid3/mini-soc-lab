# Attack Scenarios

This directory documents simulated adversary activity executed against the SOC lab environment.  
Each scenario is designed to generate realistic security telemetry for detection, investigation, and response validation.

The focus is on **controlled, repeatable attacks** that mirror real-world SOC use cases.
